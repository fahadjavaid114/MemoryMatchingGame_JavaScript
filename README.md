# MemoryMatchingGame_JavaScript

A fun, interactive browser-based memory matching game built entirely with HTML, CSS, and jQuery. I built this to practice managing game state and manipulating the DOM dynamically. 

## 🎮 How to Play
1. Click any card to flip it over.
2. Try to find the matching symbol. 
3. If they match, you'll hear a chime and they stay face-up. If they don't, you get a buzzer sound and they flip back over after a second.
4. Match all 8 pairs as fast as you can!

## 🛠️ Tech Stack & Skills Used
* **HTML5 & CSS3:** For the layout, grid styling, and flip animations.
* **jQuery / JavaScript:** For the core game logic, including event handling, dynamic card generation, and timers.
* **HTML5 Audio API:** Added sound effects for correct and incorrect matches to make the game feel alive.

## 🚀 How to Run It Locally
Since this is a vanilla frontend project, you don't need any complex build tools. 
Just clone this repository, open the folder, and double-click `index.html` to play it in your browser!
